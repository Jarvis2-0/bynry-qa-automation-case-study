## Overview
This repository contains my solution for the QA Automation case study provided by Bynry.
The goal is to demonstrate my approach to testing a multi-tenant B2B SaaS platform using
automation, API testing, and cross-platform validation.

## Tools & Technologies
- Python
- Pytest
- Playwright
- API testing using requests (conceptual)
- BrowserStack (conceptual mobile testing)

## Project Structure
- test_plan/ : Test strategy and planning
- automation/ : UI, API, and integration tests
- test_data/ : Sample test data
- reports/ : Sample test execution reports

## How to Run Tests (Conceptual)
1. Install dependencies
2. Configure environment variables
3. Run pytest commands

Note: This repository focuses on test design, structure, and automation approach.
